🌱 Agri-Tech IoT Data Pipeline & Real-Time Dashboard

📌 Project Overview

This project is a full-stack, software-simulated IoT data pipeline designed for agricultural monitoring (e.g., smart greenhouses, grain silos, cold-storage). It simulates real-time environmental sensors, streams the data to a cloud PostgreSQL database, and visualizes the metrics on a live, event-driven web dashboard.

By bypassing physical hardware, this project focuses entirely on robust data architecture, cloud database integration, and real-time frontend visualization.

🏗️ Architecture & Tech Stack

The system is composed of three main microservices:

The Edge Device Simulator (Python): Generates realistic, fluctuating environmental data (Temperature, Humidity, CO2) using random-walk algorithms.

The Cloud Database (Supabase / PostgreSQL): Acts as the central hub, securely ingesting and storing time-series data streams.

The Command Center (Streamlit): A live-polling frontend dashboard that visualizes historical trends and triggers logic-based alerts when safety thresholds are crossed.

✨ Key Features

Realistic Data Simulation: Uses controlled variations to mimic natural environmental shifts rather than erratic random numbers.

Real-Time Data Streaming: Python continuously pushes payload data to Supabase.

Live Interactive Dashboard: Streamlit UI automatically refreshes to display the latest database entries.

Dynamic Alerting System: Users can set custom thresholds via UI sliders. If real-time data crosses these limits, visual alarms are instantly triggered.

🚀 How to Run Locally

1. Prerequisites

Python 3.8+ installed on your machine.

A free Supabase account and project.

2. Database Setup (Supabase)

Run the following SQL in your Supabase SQL Editor to create the required table:

create table sensor_data (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  temperature float,
  humidity float,
  co2 float
);



3. Installation

Clone this repository and install the required Python packages:

pip install supabase streamlit pandas



4. Configuration

You need to add your Supabase credentials to the code.
Open both generator.py and app.py and replace the placeholder strings with your actual Supabase URL and API Key:

SUPABASE_URL = "your-supabase-project-url"
SUPABASE_KEY = "your-supabase-anon-key"



5. Start the System

To see the system in action, you need to run both the backend generator and the frontend dashboard simultaneously in two separate terminal windows.

Terminal 1 (Start the Edge Simulator):

python generator.py



Terminal 2 (Start the Dashboard):

streamlit run app.py
